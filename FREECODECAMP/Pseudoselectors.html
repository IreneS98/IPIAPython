Introduccion

A continiuación se expondrán las diferentes tecnolgías utilizadas durante el curso de pogramación de abril a septiembre de 2024.

En cada apartado se resumirá brevemente la tecnología y la práctica realizada.

HTML

HTML (lenguaje de marcado de hipertexto) es el lenguaje estándar para crear páginas web. HTML describe la estructura de una página web a traves de una serie de elementos.

Los elementos HTML etiquetan fragmentos del contenido y le dicen al navegador cómo mostrarlo.

Se realizó una página web sobre un blog de viajes con html.

Los HTML se almacenan en archivos .html.

CSS

CSS (hojas de estilo en cascada) es el lenguaje que utilizamos para diseñar una página web. CSS describe cómo se deben mostrar los elementos HTML, puediendo controlar el diseño de varias páginas web a la vez.

Las hojas de estilo externas se almacenan en archivos .CSS.

En el curso se utilizó el CSS para diseñar la web del blog de viajes.

JavaScript

JavaScript es el lenguaje de programación más popular del mundo, utilizado para porogramación Web.

Comparado con otros lenguajes, JavaScript es fácil de aprender. Los archivos JavaScript se almacenan como .js.

Python

Python también es un lenguaje de programación popular. Se utiliza para desarrollo web, desarrollo de software, matemáticas y sistema de scripting.

Los archivos de Python se guardan como .py.

Inteligencia Artificial

La inteligencia artificial (IA), es una disciplina y un conjunto de capacidades cognoscitivas e intelectuales expresadas por sistemas informáticos o combinaciones de algoritmos cuyo propósito es la creación de máquinas que imiten la inteligencia humana para realizar tareas, y que pueden mejorar conforme recopilen información.

La IA generativa puede ayudar a aumentar su productividad si se usa correctamente.

Algunos ejemplos de aplicaciones de IA conocida son:

  • Siri - Asistente virtual de Apple que utiliza IA para responder preguntas y realizar tareas.
  • Google Assistant - Asistente virtual de Google que utiliza IA para ayudarte en tus tareas diarias.
  • Amazon Alexa - Asistente virtual de Amazon que utiliza IA para controlar dispositivos domésticos inteligentes y responder preguntas.
  • IBM Watson - Plataforma de IA de IBM utilizada en una variedad de aplicaciones, desde el análisis de datos hasta la atención médica.
  • ChatGPT - Aplicación de chatbot de IA que se especializa en el diálogo, utilizando avanzados modelos de lenguaje para generar respuestas conversacionales.
